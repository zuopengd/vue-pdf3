# vue-pdf3
基于Vue3 + vite + pdfjs-dist版PDF预览插件

## 安装
```bash
npm install --save vue-pdf3
```

## 代码示例

### 局部引入
```vue
<template>
  <pdf url="./demo.pdf"></pdf>
</template>

<script setup>
import pdf from 'vue-pdf3'
</script>
```
